## Checklist for Windows 10

1. Enable file backups
2. Enable auto-updates
3. Security precautions
	* Disable automatic login
	* Set password requirements
	* Disable remote access
	* Disk encryption
	* Turn on the firewall
4. Use an anti-virus
	* Malwarebytes
	* Bitdefender
5. Use a cleanup tool
	* CCleaner
